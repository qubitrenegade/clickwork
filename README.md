# qbrd-tools

Reusable CLI framework for project automation.

> **Status:** Pre-1.0. API is unstable. All features are driven by real
> [orbit-admin](https://github.com/qubitrenegade/qbrd-orbit-widener) needs.

## Installation

```bash
uv pip install "git+https://github.com/qubitrenegade/qbrd-tools.git@v0.1.0"
```
